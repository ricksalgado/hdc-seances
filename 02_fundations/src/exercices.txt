* Exercício 1: Componente Funcional Básico

Enunciado:
Crie um componente funcional chamado Greeting que aceita uma prop name e renderiza uma mensagem de boas-vindas.
Por exemplo, se a prop name for "João", o componente deve renderizar "Olá, João!" .

- Exercício 2: Manipulação de Eventos e Estado
Enunciado:

Crie um componente chamado Counter que renderiza um botão e um texto indicando o número de vezes que o botão foi clicado.
Por exemplo, se o botão foi clicado 3 vezes, o texto deve mostrar "Você clicou 3 vezes no botão". Inicialize o contador com 0 e incremente-o a cada clique !

- Exercício 3: Renderização Condicional e Listas
Enunciado:

Crie um componente chamado TaskList que aceita uma prop tasks, que é um array de objetos com id e text.
Renderize uma lista ordenada de tarefas, mostrando o texto de cada tarefa. Se não houver tarefas, exiba a mensagem
"Não há tarefas a mostrar".
